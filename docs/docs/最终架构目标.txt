Persona Bot 聊天机器人
│
├── 一、基础框架（已完成）
│   ├── 核心引擎
│   │   ├── persona.py（人设加载）
│   │   ├── memory.py（记忆管理）
│   │   ├── prompt.py（Prompt 组装）

│   │
│   ├── 接入渠道

│   │
│   └── 人设系统

│

│   ├── 发牢骚和反驳
│   │   └── 修改人设 YAML 的 behavior.rules

│   ├── 情绪识别

│   │
│   └── 发送表情包

├── 三、中等功能（第二周）
│   ├── 长期记忆

│   │
│   ├── 图像识别

│   │   └── 多模态 API 调用
│   │
│   └── 文件解读
│       ├── 识别文件消息
│       └── 读取文本文件内容送入 LLM
│
├── 四、进阶功能（第三周）
│   ├── 联网搜索

│   │   └── 搜索结果注入上下文
│   │
│   ├── 成长系统

│   │   └── 亲密度随对话增长
│   │
│   └── 随机主动发消息
│       ├── 定时任务（30-120分钟随机）
│       └── 随机选择活跃用户发送
│
├── 五、高级功能（第四周）

│   │
│   └── 电话功能
│       └── VoIP 服务（难度最高，最后做）
