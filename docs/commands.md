## ⚖️ 词条权重

权重影响 AI 回复时的话题倾向，数值越高越容易提到该词。

| 指令 | 说明 |
|------|------|
| `/weight` | 查看帮助 |
| `/weight list` | 查看所有词条（按权重排序） |
| `/weight chart` | 可视化权重图表 |
| `/weight stats` | 按分类统计 |
| `/weight get <词条>` | 查看单个词条权重 |
| `/weight set <词条> <数值>` | 设置权重（0.1~10.0） |
| `/weight add <词条> <数值> [分类]` | 添加词条并指定分类 |
| `/weight del <词条>` | 删除词条 |
| `/weight search <关键词>` | 搜索包含关键词的词条 |


> 💡 权重会随聊天自动学习：你提到的词会自动加权，长期不提的词会自动衰减。

### 督学（study_guard）

无固定指令，由 AI 自动判断语义触发：

| 用户行为 | 触发效果 |
|------|------|
| 说"我去写作业了""开始学习"等 | 📋 进入督学模式 |
| 督学期间发消息（前 3 次） | 🫂 温柔劝导回去学习 |
| 督学期间发消息（超过 3 次） | 😡 骂人阶段，花式骂摸鱼 |
| 静默 10 分钟以上 | 🔄 回退到温柔劝导阶段 |
| 说"学完了""写完了"等 | 📝 退出督学，总结学习时长 |
